# HelloWorld
This simple application shows how to create an HTTP server and how to write to response.

## Requirements
* JDK 8 or later
* Maven 3.0.0 or later

## To compile
```bash
mvn celan install
```

## To run
```bash
java -jar target/helloWorldVerticle.jar
```
Or

```bash
sh run.sh
```
